#### Connection Flow for Each Platform:

##### Facebook/Instagram Connection:
```
1. Click "Connect Facebook"
2. Redirect to Facebook OAuth:
   - Login to Facebook
   - Grant permissions: manage_pages, publish_pages, read_insights
   - Select business pages to connect
3. Callback to our platform
4. Store access tokens securely
5. Sync page information and basic analytics
6. Instagram automatically connected if linked to Facebook page
```

##### LinkedIn Connection:
```
1. Click "Connect LinkedIn"
2. Redirect to LinkedIn OAuth:
   - Login to LinkedIn
   - Grant permissions: w_member_social, r_organization_social
   - Select company pages
3. Callback and token storage
4. Sync company page details
```

##### Twitter/X Connection:
```
1. Click "Connect Twitter"
2. Redirect to Twitter OAuth 2.0:
   - Login to Twitter
   - Grant permissions: tweet.read, tweet.write, users.read
3. Store API credentials
4. Sync profile information
```

  "size": "small",

```javascript
// Initiate Platform Connection
GET /api/v1/social-accounts/connect/facebook
// Returns: { "authUrl": "https://facebook.com/oauth/..." }

// Get Connected Accounts
GET /api/v1/social-accounts
// Returns: Array of connected social media accounts
```
